# Python-Packet-Sniffer-com-detector-de-ping-flood
Este projeto é um sniffer de rede escrito em Python. Ele atua interceptando o tráfego de rede e realizando o unpacking de protocolos (Ethernet, IPv4, TCP, UDP, ICMP, HTTP).   Como diferencial de segurança, possui um Módulo Ativo de Detecção de Ping Flood, que interpreta os cabeçalhos IPv4 e ICMP para alertar sobre anomalias na rede local.
